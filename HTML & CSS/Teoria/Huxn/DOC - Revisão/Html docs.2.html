Position Properties

A propriedade `position` define como um elemento é posicionado no documento. Os valores mais comuns são:

  • static: Padrão. Elementos são posicionados de acordo com o fluxo normal do documento.
  • relative: Posicionado relativo à sua posição normal.
  • absolute: Posicionado em relação ao antecessor mais próximo com `position` diferente de `static`.
  • fixed: Posicionado em relação à janela do navegador.
  • sticky: Alterna entre `relative` e `fixed` com base no scroll.
